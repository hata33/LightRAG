# LightRAG 配置（极简）

## 1. 生成配置文件
在**项目根目录**执行：
```bash
cp env.example .env
```

## 2. 必改项（编辑 `.env`）

**LLM 大模型：**
```
LLM_BINDING=openai                       # 也可 zhipuai / ollama / azure 等
LLM_BINDING_HOST=https://api.openai.com/v1
LLM_BINDING_API_KEY=你的key
LLM_MODEL=gpt-4o-mini
```

**Embedding 向量：**
```
EMBEDDING_BINDING=openai
EMBEDDING_BINDING_HOST=https://api.openai.com/v1
EMBEDDING_BINDING_API_KEY=你的key
EMBEDDING_MODEL=text-embedding-3-large
EMBEDDING_DIM=3072
```
> 换厂商/模型：只改 `*_BINDING`、`*_BINDING_HOST`、`*_API_KEY`、`*_MODEL`。
> **换向量模型必须同时改 `EMBEDDING_DIM`，并清空数据目录 `rag_storage/`（旧向量与新模型不兼容）。**

## 3. 可选项
- 服务地址：`HOST` / `PORT`（默认 `0.0.0.0:9621`）
- 认证（默认无）：`LIGHTRAG_API_KEY=xxx`，或 `AUTH_ACCOUNTS=user:pwd` + `TOKEN_SECRET=xxx`
- 数据目录：`WORKING_DIR`（默认 `rag_storage`）
- 输入目录：`INPUT_DIR`（默认 `inputs`）
