# LightRAG 启动（极简）

## 0. 前置
Python ≥3.10、`uv`、Node、`pnpm`。先按 [`配置.md`](./配置.md) 配好 `.env`。

## 1. 首次安装（仅一次）
```bash
# 后端依赖（项目根目录）
uv sync --extra api

# 前端依赖
cd lightrag_webui && pnpm install && cd ..
```

## 2. 启动（每次）
两个终端分别跑：

```bash
# 终端1 — 后端（项目根目录）
# Windows 必须加 PYTHONUTF8=1，否则启动横幅报 'gbk' 编码错
PYTHONUTF8=1 uv run lightrag-server      # → http://localhost:9621

# 终端2 — 前端（lightrag_webui 目录）
cd lightrag_webui && pnpm dev            # → http://localhost:5173
```
浏览器打开**前端地址**即可（5173 被占会自动换 5174，代理已指向后端）。

## 3. 排错
| 现象 | 原因 / 解决 |
|---|---|
| 后端报 `UnicodeEncodeError: 'gbk'` | 启动命令没加 `PYTHONUTF8=1` |
| 问答 / 文档入库失败、401 | `.env` 里 API Key 还是占位值 `your_api_key` |
| 前端能打开但无数据 | 后端没起，或 `PORT` 被改 |
| 换了向量模型后检索全乱 | 没改 `EMBEDDING_DIM` + 没清 `rag_storage/` |
